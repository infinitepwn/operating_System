# 总线

上级：[[计算机硬件简介]]

总线是计算机部件之间传送信息的公共通信通道，CPU、内存和 I/O 设备通过总线交换数据、地址和控制信号。

## 类型

- 数据总线：传送数据。
- 地址总线：指出访问的内存或设备地址。
- 控制总线：传送读写、中断、时钟等控制信号。

## 和操作系统的关系

操作系统不直接“管理总线”，但设备驱动、[[DMA]]、[[中断]]和 I/O 控制都建立在硬件总线通信之上。
