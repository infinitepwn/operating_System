# shell

上级：[[操作系统概述]]

shell 是用户和操作系统交互的命令解释器。它接收命令，启动程序，并把用户请求转换为系统调用和进程操作。

## 作用

- 解析命令行输入。
- 启动和管理进程。
- 处理重定向、管道和环境变量。
- 执行脚本。
