# IO设备

上级：[[计算机硬件简介]]

I/O 设备负责计算机与外部世界之间的数据交换，例如键盘、鼠标、显示器、磁盘和网卡。

## 分类

- 块设备：以块为单位读写，如[[磁盘]]。
- 字符设备：以字符流为单位读写，如键盘、串口。
- 网络设备：通过数据帧或数据包收发信息。
