# 缓存

上级：[[存储器]]

缓存是介于高速部件和低速部件之间的临时数据副本，用来利用局部性原理降低平均访问时间。

## 常见例子

- CPU [[高速缓存]]
- 磁盘缓存
- 文件系统缓存

## 关键思想

- 命中：需要的数据已经在缓存中。
- 未命中：需要从下一级存储中取回数据。
- 缓存行：缓存和主存之间传输的基本单位。
