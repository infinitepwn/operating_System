## 主要主题

- [概述](note/概述/概述.md)：操作系统基本概念、运行环境、系统调用、中断、处理器和内核结构。
- [进程管理](note/进程管理/进程/进程管理.md)：进程、线程、调度、进程通信、同步互斥和死锁。
- [内存管理](note/内存管理/内存管理.md)：地址空间、动态重定位、交换、空闲内存管理、虚拟内存和页面置换。
- [文件管理](note/文件管理/文件管理.md)：文件、目录、文件系统布局、磁盘空间管理和常见文件系统。
- [输入输出](note/输入输出/输入输出(IO).md)：I/O 控制方式、硬件、软件层次、DMA、设备驱动、磁盘和 RAID。
